# SplsFramework
## 简介
实现类似ET的不断完善的双端开发的游戏框架        
前端 CocosCreator       
后端 Pinus      
## 版本信息
CocosCreator    v3.2.1      
Nodejs          v9.2.0      
yarn            1.22.17     
pinus           1.4.14
